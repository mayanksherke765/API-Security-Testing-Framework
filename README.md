# Cloud Misconfiguration & API Security Scanner

A Python Flask based security assessment tool that identifies cloud misconfigurations and API security weaknesses.

## Features

- AWS S3 security checks
- IAM permission analysis
- API authentication testing
- JWT security validation
- Rate limiting analysis
- Security header checks

## Technologies Used

- Python
- Flask
- AWS Boto3
- REST API
- HTML
- CSS
- Bootstrap
